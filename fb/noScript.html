JavaScript is a dynamic computer programming language that is commonly used in browsers to enable various client-side scripts to interact with the user and alter page contents in order to create a more unique, responsive user experience.

While most of shasharala.tk is functional without JavaScript much of the content and features of each page will be unavailable or display incorrectly.


       

Please select your browser from the list above for instructions to enable JavaScript.



Google Chrome


  1. On the internet browser click the menu/"Customize and control Google Chrome" button and select "Settings".
  2. In the "Settings" menu click "Show advanced settings..." at the bottom of the page.
  3. Under the "Privacy" section click the "Content settings..." button.
  4. In the "Content settings" window, under the "JavaScript" section, select "Allow all sites to run Javascript (recommended)" and click the "Done" button.
  5. You may return to this page by clicking "Done" and closing the "Settings" tab.
  6. Click on the refresh button of the internet browser to refresh this page.

Mozilla Firefox


  1. Open a seperate tab.
  2. In the address bar type "about:config" and press Enter.
  3. Click "I'll be careful, I promise" if a warning message appears.
  4. In the search box, search for "javascript.enabled".
  5. Toggle the "javascript.enabled" preference (right-click and select "Toggle" or double-click the preference) to change the value from "false" to "true".
  6. You may close the "about:config" tab to return to this page.
  7. Click the refresh button of the internet browser to refresh this page.

Internet Explorer


  1. On the internet browser click the "Tools"/gear button and select "Internet Options".
  2. In the "Internet Options" window click the "Security" tab.
  3. Under the "Security" tab click the "Custom level..." button.
  4. In the "Security Settings - Internet Zone" window scroll down to the "Scripting" section.
  5. Select "Enable" under "Active scripting" and click "OK".
  6. When a "Warning!" window appears asking "Are you sure you want to change the settings for this zone?" select "Yes".
  7. On the "Internet Options" window select "OK".
  8. Click the refresh button of the internet browser to refresh this page.

Apple Safari


  1. On the internet browser click the "Edit"/gear button and select "Preferences".
  2. In the "Preferences" window select the "Security" tab.
  3. In the "Security" tab, under "Web content" mark the "Enable Javascript" checkbox.
  4. Click the red 'X' in the upper right corner of the "Preferences" window.
  5. Refresh this page by clicking the "reload this page" button on the internet browser.
Screenshots


Opera Software


  1. Click on the "Opera"/menu button and select "Settings".
  2. In the "Settings" tab select "Websites" from the menu on the left.
  3. Under "Javascript" select "Allow all sites to run JavaScript (recommended)".
  4. You may close the "Settings" tab by clicking the 'X' on the tab name.
  5. Click the refresh button on the internet browser to refresh this page.
